# 競艇オカルトサイン発掘ツール — 閲覧ビューア

公営競技（競艇）の**公開レース結果**から「オカルト的な条件」を統計検証した結果を閲覧するための単一HTMLビューアです。
`index.html` を開くと、検証した全サイン候補（採用/暫定/データ待ち/棄却）を一覧・絞り込みできます。

**公開URL（GitHub Pages）**: https://ohyabu0116.github.io/kyotei-occult-viewer/

## 検証の規律（なぜ信用できるか）
素朴な採掘は「小標本 × 多重比較」で偽の100%サインを量産します。これを避けるため、全候補に **4関門 + family-wise（多重比較）補正** を課しています:

1. **前半→後半の場外検証(OOS)** — 時系列前半で選抜し、未知の後半で再現するか
2. **プラセボ** — トリガーを無情報化した帰無分布と比較
3. **実配当ROI** — 実トリガ的中時の払戻で回収率を算出
4. **説明可能性** — 単なる内枠バイアスへの退化でないか

## 重要（誠実性）
- これは**予想ツールではありません**。利益は一切保証しません。
- 現時点の**確定オカルトサインは 0 件**。大半は統計的ノイズとして棄却されています。
- 一覧には実在の選手名が含まれますが、その多くは「検証した結果サインとして成立しなかった（=棄却）」という**否定的結論**です。
- データは順次拡充中です。**全データ収集後に確定結果へ自動更新**します（現在は部分データ時点のスナップショット）。

依存ゼロの自己完結HTML（外部通信なし）。
